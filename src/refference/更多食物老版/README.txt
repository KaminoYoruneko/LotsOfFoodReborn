Thanks for downloading "Lots of Food" mod, v1.12.0 for Minecraft 1.10.2.

Mod installation :
- use the installer to install Forge client
- put the mod's jar in .minecraft/mods
- launch your Minecraft using the profile "Forge"

A config file is going to spawn in .minecraft/config. It will allow you to change the generation frequency of plants and structures, along with other options.
